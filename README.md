# ZhihuFinder
## 介绍
本程序是一个可推送特别关注的人的最新动态到个人邮箱里。
## 配置
### 邮箱
```
from_addr = ""       		#发件人邮箱
pwd = ""                     #密码
to_addr = ""          #收件人
```
### 抓取对象
```
hash = ("https://m.zhihu.com/people/ivangeek","https://www.zhihu.com/people/ivangeek") #这里自行更改
```
## 运行
```
python main.py
```
